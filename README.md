# Веб-приложение для поиска и мониторинга катков Москвы

Клиентоориентированный интернет-сервис на основе открытых данных Правительства Москвы.

## Описание проекта

Веб-приложение помогает жителям Москвы находить подходящие катки, узнавать их актуальное состояние и планировать посещение с учетом загруженности.

## Технологии

- **Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5, Chart.js, Яндекс.Карты API
- **Backend:** PHP 8.0+, REST API
- **База данных:** PostgreSQL или MySQL
- **Инструменты:** Git, LaTeX

## Структура проекта

```
project/
├── frontend/          # Клиентская часть
├── backend/           # Серверная часть (API)
├── sql/              # SQL-скрипты
└── docs/             # Документация
```
