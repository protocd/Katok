## Что нужно сделать:

1. **Настроить API URL** в `frontend/js/api.js`:
   ```javascript
   baseUrl: 'http://localhost/rinks-moscow-app/backend/api'
   ```
   Измените на свой путь!

2. **Добавить Yandex Maps API ключ** (опционально):
   - Получите ключ на https://developer.tech.yandex.ru/
   - Раскомментируйте строку в `index.html` и `map.html`
   - Вставьте свой ключ

3. **Проверить работу**:
   - Откройте `http://localhost/rinks-moscow-app/frontend/index.html`
   - Проверьте все страницы
