# Веб-приложение "Катки Москвы"
## Курсовой проект, 2 курс

---

## Что это?

Простое веб-приложение для поиска катков в Москве. Пользователи могут:
- Искать катки
- Оставлять отзывы с фото
- Отмечаться на катках
- Голосовать за отзывы (вверх/вниз)
- Создавать мероприятия (если были на катке 5+ раз)

---

## Технологии (просто и понятно)

**Frontend:**
- HTML, CSS, JavaScript
- Bootstrap (для красивого вида)
- Яндекс.Карты (для карты)

**Backend:**
- PHP (простой, без фреймворков)
- MySQL (база данных)

**База данных:**
- MySQL с таблицами для пользователей, катков, отзывов и т.д.

## Как работает (простыми словами)

1. **Пользователь регистрируется** → данные в таблицу `users`
2. **Пользователь ищет каток** → данные из таблицы `rinks`
3. **Пользователь отмечается на катке** → создается `visit`, потом `checkin`
4. **Пользователь оставляет отзыв** → создается `review` с фото (опционально)
5. **Другие голосуют за отзыв** → создается `vote` (вверх/вниз)
6. **Полезные отзывы поднимаются** → сортировка по `score` (upvotes - downvotes)

---

## Основные таблицы БД (просто)

1. **users** - пользователи (email, пароль, имя)
2. **rinks** - катки (название, адрес, координаты)
3. **visits** - посещения (кто, когда, какой каток)
4. **reviews** - отзывы (текст, рейтинг, фото)
5. **checkins** - отметки присутствия (геолокация)
6. **votes** - голоса за отзывы (вверх/вниз)
7. **events** - мероприятия (догонялки и т.д.)
8. **event_participants** - участники мероприятий
